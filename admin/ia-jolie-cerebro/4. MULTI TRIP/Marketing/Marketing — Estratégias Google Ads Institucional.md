# Google Ads

> *No Google, o cliente já quer. Jolie só precisa aparecer na hora certa.*

## Por que Google Ads para Multi Trip

No Meta Ads, Jolie interrompe o scroll. No Google, o cliente JÁ está procurando.

**Intenção de compra:** altíssima. Quem digita "transfer Gramado" está pronto para decidir.

---

## Palavras-chave prioritárias

### Alta intenção (prioridade máxima)
- `transfer gramado`
- `transfer porto alegre gramado`
- `transfer aeroporto gramado`
- `transfer privativo gramado`
- `transfer poa gramado`
- `transfer salgado filho gramado`

### Média intenção
- `transfer canela gramado`
- `transfer serra gaúcha`
- `como ir de porto alegre pra gramado`
- `transporte aeroporto porto alegre gramado`

### Long tail (menor volume, maior conversão)
- `transfer privativo gramado com cadeirinha`
- `transfer executivo poa gramado`
- `transfer gramado natal luz`
- `transfer gramado julho`

## Configurações recomendadas

| Configuração | Recomendação |
|---|---|
| **Rede** | Somente pesquisa (não display junto) |
| **Geo** | RS + capitais do Sudeste (SP, RJ) |
| **Idioma** | Português |
| **Dispositivo** | Mobile em destaque (80%+ das buscas) |
| **Horário** | 6h–23h (evitar madrugada) |
| **Estratégia de lances** | Maximizar conversões → depois ajustar CPA alvo |

---

## Palavras-chave negativas

Incluir sempre para evitar cliques irrelevantes:

- grátis, gratuito, barato, econômico
- ônibus, van compartilhada
- emprego, trabalhar, vaga
- caxias, farroupilha (se não atender)
- moto, bicicleta

---

## Página de destino (landing page)

O Google Ads converte melhor para uma página específica, não para o perfil do Instagram.

**Recomendação:** criar landing page simples com:
1. Headline principal (H1): "Transfer Privativo Porto Alegre → Gramado"
2. 3 benefícios visuais
3. Botão CTA → WhatsApp
4. Preço (PIX e parcelado)
5. Fotos do veículo real
6. Avaliações reais (4.9★)
7. Logo Cadastur
