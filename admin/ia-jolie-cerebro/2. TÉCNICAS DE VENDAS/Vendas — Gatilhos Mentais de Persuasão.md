# Gatilhos Mentais

> *Use de forma natural, nunca forçada. Gatilho forçado quebra a confiança.*

## Tabela de gatilhos

| Gatilho | Quando usar | Frase modelo |
|---|---|---|
| **Autoridade** | Quando o cliente não conhece a empresa | *"A gente já faz isso todos os dias em Gramado — cada trecho, cada clima, cada detalhe."* |
| **Escassez real** | Quando a agenda está cheia de verdade | *"A agenda já está bem puxada nessa data, mas ainda consigo verificar agora."* |
| **Segurança** | Para família, CLT, primeira viagem | *"O motorista já está no aeroporto. Assim que você aterrissar ele entra em contato e te guia — você não precisa se preocupar com nada."* |
| **Facilidade** | Para qualquer perfil | *"Você não precisa se preocupar com nada. Eu organizo tudo."* |
| **Prova social** | Quando há hesitação | *"Nossos clientes chegam e já começam a aproveitar — sem stress de chegada."* |
| **Reciprocidade** | Antes de qualquer venda | Dar roteiro gratuito, dica de restaurante ou insight exclusivo ANTES de falar em venda |
| **Antecipação** | Para casal e primeira viagem | *"A viagem começa muito antes de chegar em Gramado."* |
| **Exclusividade** | Para CLT e casal | *"Carro só pra vocês, sem parada, sem dividir com ninguém."* |
| **Urgência temporal** | Para Natal Luz e inverno | *"Em julho Gramado lota — quem reserva agora garante o horário."* |

## Como NÃO usar gatilhos

❌ Criar escassez falsa ("só temos 1 vaga" quando tem várias)  
❌ Usar urgência em todo atendimento (perde o efeito)  
❌ Pressionar com vários gatilhos na mesma mensagem  
❌ Gatilho de medo sem base real  

## Sequência ideal de gatilhos por perfil

**Casal Romântico:** Antecipação → Exclusividade → Autoridade  
**Cliente CLT:** Prova social → Facilidade → Escassez  
**Família:** Segurança → Autoridade → Facilidade  
**Executivo:** Autoridade → Facilidade → (sem mais)  
**Primeira Viagem:** Reciprocidade → Antecipação → Segurança
