# Feedbacks de Clientes

> *Todo feedback é inteligência. Bom ou ruim — aprende-se com os dois.*

## Como usar este arquivo

Este arquivo é um repositório vivo. Adicionar novos feedbacks conforme chegam — via Google, WhatsApp, avaliações diretas.

**Meta atual:** 4.9★ no Google Maps

---

## Feedbacks positivos recorrentes (padrões)

Padrões identificados nos feedbacks reais de clientes:

### Pontualidade
> *"O motorista estava lá quando a gente desceu — antes de a gente procurar."*

Aprendizado: a placa com o nome no desembarque é diferencial real. Manter protocolo.

### Conhecimento da rota
> *"Ele sabia tudo sobre a estrada — foi contando sobre as curvas, os pontos da Serra."*

Aprendizado: motoristas que criam conversa sobre a rota aumentam a percepção de valor.

### Segurança com crianças
> *"Com dois filhos pequenos, o cuidado com a cadeirinha e com as curvas me deixou muito tranquila."*

Aprendizado: cadeirinha + comentários sobre segurança na serra = gatilho essencial para família.

### Comunicação antecipada (D-1)
> *"Adorei receber o nome e contato do motorista na véspera. Fiquei muito mais tranquila."*

Aprendizado: mensagem D-1 não é detalhe — é parte central da experiência.

## Template para registrar novos feedbacks

```
**Data:** [data]
**Perfil do cliente:** [Família / Casal / CLT / Executivo / Primeira Viagem]
**Canal:** [Google / WhatsApp / Indicação]
**Nota:** [1–5★]
**Resumo do feedback:** [texto]
**Aprendizado:** [o que isso ensina]
```

## Feedbacks críticos / negativos

> *Nenhum feedback negativo documentado até o momento (maio/2026).*
> Registrar aqui quando ocorrer — sem omitir. O erro documentado protege de repetir.

### Template para feedback negativo
```
**Data:** [data]
**Problema:** [descrição]
**Causa raiz:** [o que causou]
**Resolução:** [o que foi feito]
**Prevenção:** [o que mudou para não repetir]
```

## Frases de pedido de avaliação (Jolie)

Jolie envia mensagem gentil pós-viagem para solicitar avaliação no Google:

**Versão padrão:**
> *"Oi [nome]! Espero que a viagem de vocês tenha sido incrível 🍷 Se quiser compartilhar a experiência com o transfer, ficamos muito felizes com uma avaliação no Google — é o que nos ajuda a continuar cuidando de cada detalhe. [link] Obrigada!"*

**Versão família:**
> *"Oi [nome]! Espero que as crianças tenham adorado a Serra ☃️ Se o transfer foi bom pra vocês, uma avaliação no Google nos ajuda muito. [link] Até a próxima viagem!"*

**Versão casal:**
> *"Oi! Espero que a viagem de vocês tenha sido tudo que imaginaram 🍷 Ficamos felizes em ter sido parte desse momento. Se quiser deixar uma avaliação, será muito especial pra gente. [link]"*
